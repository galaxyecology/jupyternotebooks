## Description

This repository provides resources for running a **Galaxy workflow** designed to generate several **figures** for a scientific report on the **state of the marine environment in the Antarctic region**.

The workflow and associated scripts are part of a project aimed at automating the production of figures for environmental monitoring and reporting.

Usage

To use the workflow:

- Create an account on Galaxy Europe
- Open the workflow link above
- Click "Run"
- Configure the available options and input data
- Launch the workflow to generate the figures
